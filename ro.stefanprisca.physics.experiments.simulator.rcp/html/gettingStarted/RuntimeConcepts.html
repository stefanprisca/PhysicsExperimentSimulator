Runtime Concepts

These are some things that you need to keep in mind when editing and running experiments using this software:

  1. Equations are computed in the order that they are written! The result of the last equation is considered to be the result of the experiment.
  2. Experiment variables are marked with surrounding '{}'. e.g.:'{a}'
    \t Note: if you do not surround the variables with '{}' than the parser will not know how to compute them.
  3. The next special variables are pre-defined:
    1. {PI}
    2. {E}
    3. {G}

    Some more variables will be defined in the next releases (e.g.: light-speed, sound-speed, density of different materials, etc)
  4. All variables used in the equations must also be declared in the variable section!
    The equation editor will ensure there are no undeclared variables and will ask if you want to add the undeclared ones. It is highly recommended to do so!
  5. You can use methods from java.lang.Math class to compute your experiments.
    There is a button in the equation editor dialog that allows you to look through this methods.
    \t Note: After inserting a method through the dialog, you need to complete the variable fields!
  6. Numbers used in equation MUST be in decimal form! e.g.: 2.0 and NOT 2

Beside the above, there is not much to say. Keep in mind that the software may still be buggy.
