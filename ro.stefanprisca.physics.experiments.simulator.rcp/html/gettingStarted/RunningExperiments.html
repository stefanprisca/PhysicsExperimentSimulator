Running Experiments

After having the experiments displayed in the list and double clicking on one, you can run them.

There are two ways of running an experiment:

  • A single time
  • For a period T
    (Here you need to provide the period in the field below)

View the results

In order to view the results you need to open the Results console.
This can be done from the console view, in the following way: from the bottom right corner select Open Console -> Results

If there are problems at runtime, this will appear in the Problems View
