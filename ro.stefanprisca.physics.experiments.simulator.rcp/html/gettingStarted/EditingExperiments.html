Editing Experiments
Experiments can be edited via the Experiment Editor.

In order to edit experiments, select one from the list and press the edit button. At this point, the experiment editor will pop up in the editor area.
This editor allows you to edit every field of the experiment. In order to modify equations and variables, there are two expanding sections in the right part of the editor.
When editing equations and variables keep in mind the points presented in the Runtime Concepts section. There are the Move Eq Up and Move Eq Down buttons which allow you to order equations in the desired way.
