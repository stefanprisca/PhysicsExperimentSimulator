Some Mathematical examples

  • Computing the 2nd order equation:
    {delta} = pow({b},2.0) - 4.0*{a}*{c}
    {x} = ({b} - sqrt({delta}) ) / 2.0*{a}
    {x} = ({b} + sqrt({delta}) ) / 2.0*{a}
     You can see that the math functions pow and sqrt are used here.
  • Computing the nth Fibonacci number can be done in the following way:

    (1) set-up the following equations in the experiment:

    {fib} = {fib1} + {fib2}
    {fib2} = {fib1}
    {fib1} = {fib}
    (2) set the initial values for variables like this: fib1 = 1, fib2 = 0

    (3) run the experiment for n times to get the nth Fibonacci number.
